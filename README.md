# Proxy IP Changer

Скрипт для автоматической смены IP адреса через сеть Tor с возможностью включения/выключения прокси.

## Требования

### Необходимое ПО

1. **Homebrew** - менеджер пакетов для macOS
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

2. **Python 3** и **venv**
   ```bash
   # Обычно уже установлен в macOS
   python3 --version
   ```

3. **Tor** - сеть анонимности
   ```bash
   brew install tor
   ```

4. **Python зависимости** (устанавливаются автоматически при запуске)
   ```bash
   requests>=2.31.0
   ```

## Установка

1. Клонируйте или скачайте проект:
   ```bash
   cd /path/to/proxyIPChange
   ```

2. Создайте виртуальное окружение:
   ```bash
   python3 -m venv venv
   ```

3. Активируйте виртуальное окружение:
   ```bash
   source venv/bin/activate
   ```

## Использование

Запустите главный скрипт:
```bash
./start.sh
```

### Меню:

1. **Включить прокси** - включает SOCKS прокси через Tor и запускает бесконечную смену IP
2. **Выключить прокси** - отключает прокси и останавливает Tor, восстанавливая обычное интернет-соединение
3. **Запустить IP-менеджер** - запускает смену IP на указанное количество минут
4. **Выход** - завершение работы

## Как это работает

- **Прокси**: SOCKS5 на `127.0.0.1:9050`
- **Сеть**: Tor для анонимизации
- **Интервал смены IP**: 10 секунд
- **Ожидание Tor**: 15 секунд после перезапуска

## Файлы

- `start.sh` - главный скрипт управления
- `change_ip.py` - скрипт смены IP адреса
- `venv/` - виртуальное окружение Python

## Устранение проблем

### Ошибка `ERR_PROXY_CONNECTION_FAILED`
- Выберите опцию 2 для отключения прокси
- Проверьте, что Tor остановлен: `brew services list | grep tor`

### Медленная работа интернета
- Tor работает медленно по своей природе
- Увеличены таймауты для стабильной работы
- Рекомендуется использовать для задач, не требующих высокой скорости

### Tor не запускается
```bash
brew services restart tor
brew services list | grep tor  # проверка статуса
```
